# MyHeroesDB

MyHeroesDB is a website to gather information from over 500 characters around the universe.

## Installation

```Locate the folder
pip install -r requirements.txt
```

## Usage
```Create a virtual environment
virtualenv env
env\scripts\activate

```flask
flask run
